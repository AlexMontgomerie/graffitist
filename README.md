## Static Quantisation

The static quantisation method requires the following information.

 - Model Directory
 - Input Graph
 - Optimised Graph
 - Quantised Graph
 - Input Node
 - Output Node
 - Input Shape
 - Quantisation Widths

```
./quantise_static.sh
```
